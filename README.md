## Features

- Real-time one-to-one chat.
- Real-time message status updates and read receipts.
- Pin and unpin chat.
- Archive and unarchive chat.
- Clear messages and delete chat.
- Light and dark mode.

> This project is a WIP and more features will be added soon.
